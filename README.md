# E-commerce Data Pipeline

## 🚀 Overview
Built an end-to-end ETL pipeline to process e-commerce data and generate business insights.

## ⚙️ Architecture
CSV → Extract → Transform → Load → SQL → Insights → Parquet

## 🛠 Tech Stack
- Python (Pandas, NumPy)
- SQLite
- Parquet

## 📊 Features
- Data cleaning & validation
- Feature engineering (total_price)
- SQL-based analytics
- Data stored in columnar format (Parquet)

## 📈 Sample Insights
- Total Revenue: 126000
- Top Products: Phone, Headphones
